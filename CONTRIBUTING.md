# Contributing to ORBIT-X

First off, thank you for considering contributing to ORBIT-X! It's people like you that make ORBIT-X such a great tool for autonomous satellite monitoring.

## How Can I Contribute?

### Reporting Bugs
* Check the existing issues to see if the bug has already been reported.
* If you can't find an open issue that describes the problem, open a new one. Include as much information as possible, including steps to reproduce the bug.

### Suggesting Enhancements
* Open a new issue and describe the enhancement you'd like to see.
* Explain why this enhancement would be useful to other users.

### Pull Requests
1. Fork the repository.
2. Create a new branch for your feature or bug fix.
3. Make your changes and commit them with descriptive messages.
4. Push your changes to your fork.
5. Submit a pull request to the `main` branch.

## Technical Stack
- **Frontend**: Electron, Vanilla JS, CSS3
- **Backend**: Node.js, Python (Neural Core)
- **Database**: MS Access (Local Edge), ThingsBoard (Cloud)
- **Hardware**: ESP32, MQTT

## Style Guide
- Use consistent indentation (4 spaces).
- Write descriptive comments for complex logic.
- Ensure all new features are documented in the `README.md`.
